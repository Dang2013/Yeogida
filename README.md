# 당신에게 딱 맞는 여행지 추천 - 여기다

---

## 프로젝트 개요

---

국내 관광객들에게 빅데이터를 이용한 추천 서비스 구축

## 프로젝트 정보

---

코로나 19가 주춤해짐에 따라 국내 관광객 수요가 증가하고 있다. 이러한 수요에 맞춰 한국관광데이터랩 데이터와 공공데이터포털, 기상청 데이터 등을 토대로 빅데이터 추천 알고리즘을 구축, 사용자들에게 관광지를 추천하는 서비스 이다.
